## 目录结构

### 主要文件

- 主文件入口：main.tex
- 参考文献 bib：main.bib
- A1 - A4
  - A1 中文封面
  - A2 英文封面
  - A3 原创性声明 + 著作权声明
  - A4 答辩委员会名单
- B1 - B2
  - B1 中文摘要
  - B2 英文摘要
- C1 - Cx 正文部分
- D2 - D3
  - D2 致谢
  - D3 学术成果说明



### 其他目录

- resources 存放论文中的图片等文件（也可自行更改）
